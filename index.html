Benchmarking Open Event Extraction with a Large-scale Chinese Title Dataset.

What is Title2Event?

Title2Event is a open event extraction dataset with large-scale human annotated Chinese title. Title2Event contains more than 42,000 news titles in 34 topics collected from Chinese web pages. It is collected by researcher at Harbin Institute of Technology and QQ Browser Search.(最后我们把数据集扩充到了xxx条)

What is MAVEN?

Title2Event is a event detection dataset with massive general domain human annotated data. It contains 4,480 documents; 118,732 event mentions and 168 event types. It is collected by researcher at Tsinghua University and WeChat AI.

For more details, please refer to our EMNLP 2020 paper:

(Wang et al., 2020)
